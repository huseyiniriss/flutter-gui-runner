# Phase 06 — Dependencies & projects

## Dependencies
- Parse `pubspec.yaml`; list dependencies / dev_dependencies with versions.
- `flutter pub outdated --json` → show upgradable, one-click upgrade.
- Add package: search pub.dev API (`https://pub.dev/api/search?q=`), pick,
  `flutter pub add <pkg>`.
- Remove package: `flutter pub remove <pkg>`.

## Projects
- Multi-project switcher (already scans `~/Documents/projects`).
- Recents + favorites; "Open in editor" (Zed: `zed <path>`).
- Per-project remembered device/mode/flavor.
- FVM version shown per project.
